About

The User Manager app is a simple demo project that shows how to build a CRUD (Create, Read, Update, Delete) system using only HTML, CSS, and JavaScript.

It lets you add new users, edit their information, delete users, and search through the list quickly. All the data is stored in your browserâ€™s LocalStorage, so even if you refresh the page, your user list is saved.

This project is a great way to practice important web development concepts like form handling, table rendering, JavaScript validation, and responsive design.
